 chatbot-in-python
Este proyecto es un chatbot interactivo creado en Python. El bot es capaz de conversar con el usuario, ejecutar comandos especiales, recordar información temporal durante la sesión y aprender nuevas respuestas que el usuario le enseñe. Su diseño es sencillo, ideal para comenzar en el mundo de la programación y entender cómo funcionan los bots conversacionales.
